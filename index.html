Event Row
The app lists your calendar events in chronological order. Each event row summarizes the event and reveals details when it's tapped.
Minimum Event Row Contents
  • Remaining Days, Date and Time, Title, Calendar Color
Other Contents
  • Note, Location, Travel Time, Alert(s), Event Recurrence
Expanded Mode
  • Shows details of the contents above
  • Edit button to open the event editor
  • A map if the location has position information. You can open iOS Maps or Google Maps to get a direction quickly
